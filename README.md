# 📝 To-Do List Application (C++)

This is a console-based To-Do List application developed in C++ as part of CodeAlpha Internship.

## 🚀 Features
- Add tasks
- View tasks
- Mark tasks as completed
- File saving/loading

## 💻 Technologies Used
- C++
- File Handling
- OOP Concepts

## 📂 How to Run
1. Compile the code:
   g++ main.cpp -o todo
2. Run:
   ./todo

## 📸 Output
Console-based task manager
